Sistema de Agendamento de Salas

Sistema web para controle e gerenciamento de salas

Sobre

Introdução

Este projeto consiste em um sistema de agendamento de salas desenvolvido para fins acadêmicos em um curso técnico. Sua finalidade principal é administrar eficazmente os laboratórios e salas de instituições de ensino. Para alcançar esse objetivo, o sistema oferece diversas funcionalidades e uma interface web responsiva e dinâmica.

Desenvolvimento

O sistema foi desenvolvido utilizando PHP, com as interfaces construídas em HTML e CSS, beneficiando-se do auxílio do framework Bootstrap para conferir à interface uma maior dinamicidade, interatividade e responsividade. Isso resulta em uma experiência de usuário aprimorada. Além disso, o desenvolvimento desse sistema coloca em prática conceitos fundamentais, como o relacionamento de entidades em bancos de dados, operações de CRUD e princípios de User Experience (UX). Para a gestão de dados, optou-se pelo SGBD MySQL.

Home

Na tela principal (Home), os usuários podem verificar a disponibilidade das salas por meio de um calendário que exibe os agendamentos armazenados no banco de dados para a sala escolhida. Isso proporciona uma representação visual dos agendamentos, tornando o uso mais dinâmico. Ao clicar no botão 'Agendar' ou em um dia no calendário, os usuários serão direcionados para a tela de agendamento.

Agendamento

O processo de agendamento de salas é realizado por meio de um formulário que inclui validações e regras de negócio essenciais para evitar inconsistências nos agendamentos e garantir a integridade das informações. Este formulário é preenchido com detalhes, como o nome da sala ou laboratório, data e horário do agendamento, e outros dados relevantes.

Os agendamentos podem ser configurados de duas formas: individualmente ou como repetições semanais. Além disso, os usuários têm a flexibilidade de agendar salas para si mesmos, enquanto administradores têm a capacidade de agendar salas em nome de outras pessoas. Também é possível adicionar comentários e solicitar equipamentos necessários durante o processo de agendamento.

Relatórios

Os administradores têm acesso a uma seção na qual podem criar relatórios de agendamentos, fazendo uso de filtros ou pesquisando agendamentos específicos. Isso possibilita uma gestão mais eficiente da utilização das salas e um controle mais preciso sobre elas. Além disso, eles têm a capacidade de visualizar, editar e excluir agendamentos.

usuários

O sistema oferece três níveis de acesso. Inicialmente, os visitantes têm acesso apenas à tela inicial (Home), onde podem verificar os agendamentos existentes. No entanto, para desfrutar de todas as funcionalidades do sistema, é necessário realizar o cadastro e efetuar o login.

Os três tipos de acesso no sistema são os seguintes:

  • Aluno - o nível mais básico, que oferece funcionalidades essenciais.
  • Professor - permite solicitação de equipamentos, agendamentos em massa e tem a capacidade de sobrepor agendamentos de alunos, notificando-os por e-mail sobre a realocação de seus agendamentos devido à prioridade do professor.
  • Administrador - tem acesso a recursos avançados, como a geração de relatórios, a edição e exclusão de agendamentos, a possibilidade de agendar salas em nome de outras pessoas e a administração dos usuários cadastrados.
